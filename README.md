# Elastic ASCII Playground

An interactive web playground for creating and manipulating ASCII art with physics-based animations.

## Features

- **Image to ASCII Converter**: Upload images and convert them to ASCII art
- **Interactive Canvas**: Drag and manipulate ASCII characters with spring physics
- **Manual Input**: Type or paste your own ASCII art
- **Zoom Controls**: Zoom in/out to see details

## How to Use

1. **Image Converter Tab**:
   - Click "Choose Image" to upload an image
   - Click "Convert to ASCII" to transform it
   - Click "Send to Canvas" to add it to the interactive view

2. **Manual Input Tab**:
   - Type or paste ASCII art
   - Click "Add to Canvas" to visualize it

3. **Interactive Canvas**:
   - Drag any character to stretch connections
   - Release to watch it spring back
   - Press 'R' to reload/parse ASCII
   - Press 'L' to lock/unlock nodes

## Deployment

This is a standalone HTML file with no dependencies. All fonts load from Google Fonts and p5.js loads from CDN.

Simply upload `index.html` to any web hosting service:
- GitHub Pages
- Netlify
- Vercel
- Any static hosting
